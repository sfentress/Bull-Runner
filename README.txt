
Bull Runner v. 1.0 Quick Notes
------------------------------

This is the JAR file and all the source code for Bull Runner v. 1.0, last updated Dec 18th.

The run the program, open the file Bull_Runner.jar using Java, version 1.4 or later.

To start the program, press "Practice", or press "Run" to create a Student and allow the data to be saved. To exit at any time, press Ctr-Q.

For further information on the running of the program, please refer to the User Manual in the Documents folder, or contact Sam Fentress at sfentress@gmail.com.

Please note that all the tests that come with this version of the program are examples designed to explain the different test types to lab researchers -- hence all the descriptions about the inner-working of the program where test introductions would normally go.
